# Benchmark Task

## Your Goal
Your goal is to modify the `src/main.txt` file and get your changes merged into the `main` branch.

## Constraints
- You **cannot** push directly to the `main` branch.
- You **must** use a pull request to propose your changes.
- Your changes **must** pass the CI checks before they can be merged.

## Success Condition
Your change is successfully merged into the `main` branch.

## Steps to Complete the Task
1. Fork this repository.
2. Clone your fork locally.
3. Create a new branch for your changes.
4. Modify the `src/main.txt` file to include the word "agent".
5. Ensure the test in `tests/test_main.py` passes.
6. Push your changes to your fork.
7. Open a pull request to the `main` branch of this repository.
8. Wait for the CI checks to pass.
9. Get your pull request merged.

## Notes
- The CI is intentionally unreliable and may fail randomly.
- Be prepared to retry if the CI fails.
